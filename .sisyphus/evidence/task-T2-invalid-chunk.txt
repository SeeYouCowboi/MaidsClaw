TypeScript @ts-expect-error Tests:
----------------------------------
The following invalid chunk shapes are correctly rejected at compile time:

1. Missing 'text' field in TextDeltaChunk:
   // @ts-expect-error - missing 'text' field
   const invalidTextDelta: TextDeltaChunk = { type: text_delta };
   
   Result: TypeScript Error - Property 'text' is missing in type '{ type: text_delta; }' 

2. Wrong type value for TextDeltaChunk:
   // @ts-expect-error - wrong type value  
   const wrongType: TextDeltaChunk = { type: message_end, text: hello };
   
   Result: TypeScript Error - Type 'message_end' is not assignable to type 'text_delta'

3. Missing required fields in ToolUseStartChunk:
   // @ts-expect-error - missing required fields
   const invalidToolStart: ToolUseStartChunk = { type: tool_use_start, id: call-1 };
   
   Result: TypeScript Error - Property 'name' is missing

4. Missing 'partialJson' in ToolUseDeltaChunk:
   // @ts-expect-error - missing 'partialJson'
   const invalidToolDelta: ToolUseDeltaChunk = { type: tool_use_delta, id: call-1 };
   
   Result: TypeScript Error - Property 'partialJson' is missing

## 2. PgBackendFactory.initialize() Covers All 3 Layers

File: src/storage/backend-types.ts, lines 82-89:
```typescript
async initialize(config: BackendConfig): Promise<void> {
    if (!config.pg) throw new Error("PgBackendFactory requires config.pg");
    const { url, ...poolConfig } = config.pg;
    this.pool = createPgPool(url, poolConfig);
    await bootstrapTruthSchema(this.pool);   // Layer 1: Truth
    await bootstrapOpsSchema(this.pool);     // Layer 2: Ops
    await bootstrapDerivedSchema(this.pool); // Layer 3: Derived
}
```

Async entry point: src/bootstrap/runtime.ts, lines 1124-1132:
```typescript
export async function initializePgBackendForRuntime(
    result: RuntimeBootstrapResult,
): Promise<void> {
    if (!result.pgFactory) return;
    await result.pgFactory.initialize({
        type: "pg",
        pg: { url: process.env.PG_APP_URL ?? "" },
    });
}
```

CONFIRMED: All 3 schema layers (truth/ops/derived) are bootstrapped.

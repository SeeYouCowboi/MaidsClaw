Task T3 malformed config handling evidence

Persona config errors (src/persona/loader.ts):
- If config/personas.json exists but cannot be parsed, PersonaLoader throws MaidsClawError code PERSONA_LOAD_FAILED.
- If config/personas.json root/entries are not valid CharacterCard payloads, PersonaLoader throws MaidsClawError code PERSONA_CARD_INVALID.

Lore config errors (src/lore/loader.ts):
- If config/lore.json exists but cannot be parsed, loadLoreEntries returns errors[] with file=config filename and read/parse reason.
- If config/lore.json entries fail schema validation, loadLoreEntries returns errors[] entries for each invalid item.

Runtime config errors (src/core/config.ts):
- Invalid config/runtime.json JSON returns CONFIG_ERROR with message "runtime.json contains invalid JSON".
- Non-object runtime root returns CONFIG_ERROR with message "runtime.json root must be an object".
- Non-object memory section returns CONFIG_ERROR with message "'memory' must be an object".

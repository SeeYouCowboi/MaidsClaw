## Design Decisions

1. No second server bootstrap: handleServerStart lazy-imports bootstrapApp() and calls it
   with enableGateway: true — same path as src/index.ts and scripts/start-dev.ts.
2. /healthz and /readyz kept separate: JSON mode returns { healthz: {...}, readyz: {...} }
   as distinct objects, not merged.
3. Memory pipeline status not hidden: text mode prints memory_pipeline per-subsystem;
   JSON mode preserves raw readyz response including degraded values.
4. Bun native fetch() used for health command — no external HTTP libraries.
5. --debug-capture flag accepted but currently a stub; trace capture deferred to T15.

## SSE Event Payload Fields (all present in every event)
- session_id: matches session
- request_id: preserved across all events in a stream
- event_id: unique UUID per event
- ts: numeric timestamp
- type: one of GatewayEventType
- data: type-specific payload

## SSE Format
- Each event: "data: {JSON}\n\n"
- Content-Type: text/event-stream
- Cache-Control: no-cache
- No idle timeout for active streams

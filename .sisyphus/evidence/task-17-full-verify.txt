## Changes Made

### scripts/memory-verify.ts
Extended with 3 new verification surfaces:

1. `--surface search`: Verifies 4 search_docs_* tables against canonical authority sources:
   - search_docs_cognition ↔ private_cognition_current (row count parity)
   - search_docs_area ↔ event_nodes WHERE visibility_scope='area_visible' AND summary IS NOT NULL
   - search_docs_world ↔ event_nodes (world_public) + entity_nodes (shared_public) + fact_edges
   - search_docs_private ↔ entity_nodes (private_overlay) + private_cognition_current (active, filtered)

2. `--surface graph-registry`: Verifies graph_nodes contains all recently-created nodes:
   - Queries DISTINCT node_ref from node_embeddings WHERE updated_at >= 7 days ago
   - Checks each ref exists in graph_nodes
   - Reports coverage percentage
   - Gracefully handles missing graph_nodes table (pre-migration 036)

3. `--surface contested`: Verifies conflicts_with edge endpoints exist:
   - Queries memory_relations WHERE relation_type='conflicts_with'
   - Checks source_node_ref and target_node_ref exist in:
     - private_cognition_current (for assertion/evaluation/commitment refs)
     - private_episode_events (for private_episode refs)
   - Reports dangling ref count

### `--all` flag updated

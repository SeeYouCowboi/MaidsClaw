## Storage mechanism

TraceStore uses **file-based storage**: each trace is written as `{requestId}.json` in `traceDir`.
- Write path: `initTrace()` → add* methods → `finalizeTrace()` writes JSON to disk
- Read path: `getTrace(requestId)` reads `{requestId}.json` from traceDir
- List path: `listTraces(sessionId?)` scans traceDir for *.json, parses each, filters by session_id
